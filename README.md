Peer-reviewed project for Getting and Cleaning Data
Course by John Hopkins University
coursera.org

## What's Included

1. a tidy data set, file: _tidy-av-activity-data.txt_
2. a code book that describes the variables, the data, and any transformations performed to clean up the data, file: _CodeBook.md_
3. code for performing the analysis, file: *run_analysis.R*. The code provides all the steps for downloading and unzipping raw data files into your working directory and for transforming the data to obtain the final tidy data file
